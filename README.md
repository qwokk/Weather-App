# Weather App (Приложение погоды)

#### Простое и элегантное веб-приложение для отслеживания погоды в реальном времени. Проект выполнен на чистом JavaScript (Vanilla JS) с использованием современных API.

# Стек технологий

#### HTML5: Семантическая структура документа.
#### CSS3: Flexbox верстка, кастомные шрифты (Poppins) и современная стилизация.
#### JavaScript (ES6+): Асинхронные запросы (fetch, async/await), работа с DOM и объектная логика маппинга иконок.
#### API: Open-Meteo API (бесплатное и не требует ключа).

# Основной функционал
#### Динамический поиск: получение координат любого города через Geocoding API.
#### Данные в реальном времени: отображение температуры, влажности и скорости ветра с помощью Open-Meteo API.
#### Адаптивные иконки: автоматическая смена графики в зависимости от погодных условий (солнечно, дождь, снег и т.д.).
#### Современный UI: чистый адаптивный интерфейс, вдохновленный современными трендами дизайна.

# Структура проекта
#### index.html — разметка интерфейса и подключение ресурсов
#### style.css — стилизация приложения, включая Flexbox-сетку и адаптивность
#### script.js — логика обработки ввода и взаимодействие с QR API
